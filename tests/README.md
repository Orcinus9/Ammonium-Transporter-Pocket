# Testing Plan

## Purpose
This folder will contain test cases to check whether the program accepts valid sequences, rejects invalid ones, and produces expected outputs.

## Version 1 tests
- Test a valid amino acid sequence
- Test an invalid sequence with unsupported characters
- Test an empty input
- Test whether the system returns a pocket prediction result
- Test whether the system returns a transport-effect explanation

## Notes
Early tests may use mock outputs before real structure and pocket prediction tools are connected.
