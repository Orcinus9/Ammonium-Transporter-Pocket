# Frontend Plan

## Purpose
The frontend will allow the user to enter an amino acid sequence and view the predicted binding pocket and transport-effect explanation.

## Version 1 features
- Text box for amino acid sequence input
- Submit button
- Result area for sequence validation
- Result area for predicted pocket output
- Result area for ammonium transport interpretation

## Notes
Version 1 should focus on a simple and clear interface before adding advanced visualizations.
