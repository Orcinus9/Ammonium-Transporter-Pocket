# Backend Plan

## Purpose
The backend will handle sequence validation, structure retrieval or modeling, pocket prediction, and transport-effect interpretation.

## Version 1 tasks
- Accept amino acid sequence input
- Validate sequence format
- Identify whether the sequence may belong to an ammonium transporter-related protein
- Connect sequence to structure data or a future structure model
- Predict a candidate ligand-binding pocket
- Return a simple explanation of possible impact on ammonium transport

## Notes
Version 1 will begin with a simple and explainable workflow before adding advanced prediction tools.
